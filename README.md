## how to use
* ログインするかはじめてかを尋ねられるので、どちらかを選んでください
* どっちにしても、メールアドレスとパスワード入力してください
* ログインできたら、これまでに追加されたメッセージがみられると思います
* あとは、適当に、テキストボックスに入力して、Sendすれば、メッセージが送れる。
